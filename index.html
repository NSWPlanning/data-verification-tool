Creating a new Council

  1. Create a new Council
  2. Create a user for the Council.
  3. Upload data files (see section 'Manually upload Council data files').

Editing and Updating users

To edit a user: 1. Click on Admin | Users. 2. Edit the name, email, password, or permissions. 3. Save the user.

To edit a Council definition: 1. Click on Admin | All Councils to return to the homepage. 2. Start to type the name of the Council in the textbox named 'Filter by Council name ...' 3. When you see the name of the Council you are interested in, click on it.
